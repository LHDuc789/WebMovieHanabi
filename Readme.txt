-Vào thư mục có chứa file manage.py (đường dẫn : Web\PJReview\pjreview)
Chạy các lệnh sau:
pip install keras-preprocessing

pip install django-crispy-forms

pip install dill

python manage.py makemigrations

python manage.py migrate

python manage.py collectstatic (dòng này nếu nó kêu
gõ yes hoặc no thì gõ yes, còn nếu không kêu thì thôi)

py manage.py runserver
